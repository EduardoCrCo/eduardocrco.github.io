Around the U.S.

Project Description

Around the U.S. is an interactive, full-stack web application that allows users to share and discover photographs of places across the United States. The application implements a complete photo-sharing social networking system with user authentication, profile management, and social interactions through a "like" system.

Features

  • Authentication System: User Registration and Login JWT-Based Authentication (JSON Web Tokens), Robust Validation, Private Path Protection, Session Persistence in LocalStorage.
  • Card/Photo Management: users Can create, delete, view.
  • Update, edit and viewing users profile and avatar.
  • Data validation, input sanitization, password encryption, Safety Headers with Helmet, Rate limiting to prevent attacks, CORS configured for production.
  • Structured logging system with Winston HTTP request and error logging Centralized error handling with custom middleware MongoDB connection logs.

Key Achievements

  • Robust Architecture: Clear Frontend/Backend Separation with RESTful APIs, MVC Pattern Implemented in the Backend (Models, Views, Controllers), effective componentization in React with modern hooks, context API for Managing Global User State.
  • Advanced Refactoring: Centralization of validations in dedicated middleware Consistent error handling with custom classes HTTP response helpers for API consistency Separation of responsibilities in controllers.
  • Production Deployment: Frontend deployed on Netlify with Vite automatic build Backend deployed on Railway with PM2 for process management MongoDB database in the cloud Environment variables configured for different environments.
  • User Experience: Smooth navigation with React Router v6 Visual feedback with informative tooltips Load state and error handling Responsive interface with modular CSS.

Summary

Around the U.S. represents a complete full-stack project that demonstrates mastery of modern web technologies. The application combines a robust backend architecture based on Node.js/Express with a reactive user interface built in React.

The project stands out for its focus on best development practices, including thorough data validation, professional error handling, structured logging, and production deployment. The documented refactoring demonstrates a commitment to code quality and long-term maintainability.
