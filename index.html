De Drones

Project Description

A full-stack video management platform built with modern web technologies. The application features a React frontend with Vite and a robust Node.js/Express backend, enabling users to manage videos, create playlists, and interact through a secure authentication system.

Key Features

  • Full-Stack Architecture: React + Vite frontend with Node.js/Express backend
  • Secure Authentication: JWT-based user registration and login system
  • Video Management: Integrated with YouTube Data API v3 for video search
  • Database Integration: MongoDB for storing user data, videos, and playlists
  • Dashboard Analytics: Interactive visualizations and metrics system
  • Responsive Design: Optimized for desktop and mobile devices

Summary

This project demonstrates my full-stack development expertise, showcasing seamless integration of modern technologies, third-party APIs, and production-grade security to deliver a scalable video management solution.

Alrededor de México

Project Description

An interactive photo-sharing social platform where users can share and discover photographs of places across México. Features complete user authentication, profile management, and social interactions through a like system.

Key Features

  • JWT Authentication: Secure user registration, login, and session persistence
  • Photo Management: Create, delete, and view photo cards with like functionality
  • Profile System: Update user profiles and avatar management
  • Production Security: Data validation, password encryption, rate limiting, and CORS
  • MVC Architecture: RESTful APIs with centralized error handling and middleware
  • React Frontend: Modern hooks, Context API, and responsive design

Summary

A complete full-stack social platform showcasing modern web development best practices, from robust backend architecture with Node.js/Express to responsive React frontend, emphasizing code quality and production-ready deployment.

De Patria a Patria

Project Description

An interactive web art gallery showcasing stories of global places through photographs and narratives from technology professionals. Features fully responsive design and integrated NFT functionality for digital art purchases.

Key Features

  • Responsive Design: Adapts from 320px mobile to 1280px desktop with optimized breakpoints
  • Interactive Gallery: Adaptive image grid with smooth transitions across devices
  • BEM Methodology: Modular CSS architecture for maintainable and scalable code
  • Performance Optimized: Fast loading with WOFF2 fonts and web-optimized images
  • NFT Integration: Blockchain connections for digital art marketplace functionality
  • Professional Design: Inter typography with effective dark color scheme

Summary

A modern web gallery showcasing exemplary HTML5/CSS3 implementation with BEM methodology, achieving perfect balance between functionality and aesthetics while ensuring optimal experience across all devices.
